# HAI-API Web

企业AI模型网关与令牌管理系统 - 前端界面

## 技术栈

- React 18
- React Router
- Tailwind CSS
- Lucide React Icons
- Axios

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── pages/          # 页面组件
├── components/     # 公共组件
├── services/       # API服务
├── App.jsx        # 应用入口
└── index.css      # 全局样式
```

## 页面

- `/login` - 用户登录
- `/register` - 用户注册
- `/dashboard` - 控制面板
- `/channels` - 渠道管理
- `/tokens` - 令牌管理

## 设计风格

- Material Design 3
- 渐变配色（蓝色到靛蓝色）
- 圆角卡片设计
- 平滑过渡动画
- 响应式布局
